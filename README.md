## 📌 Overview

The **AI Smart Study Platform** is a modern web application that enhances student learning using AI-driven tools and real-time analytics.

It helps students:
- 📚 Organize study materials  
- 🧠 Generate AI summaries  
- 🃏 Create flashcards  
- 📊 Track progress  
- 🤖 Predict exam questions
